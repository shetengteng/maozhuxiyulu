## 设计系统

### 配色方案

| 角色 | 色值 | 说明 |
|------|------|------|
| Primary (红) | `#DC2626` | 中国红，主色调点缀 |
| Primary Dark | `#991B1B` | 深红，hover 状态 |
| Background | `#FAFAF9` | 暖白底色 |
| Surface | `#FFFFFF` | 卡片白 |
| Text Primary | `#1C1917` | 主文字深色 |
| Text Secondary | `#57534E` | 副文字灰 |
| Text Muted | `#A8A29E` | 辅助信息 |
| Border | `#E7E5E4` | 边框线 |
| Gold Accent | `#CA8A04` | 金色点缀（星标等） |

### 字体

- **标题/语录**: Noto Serif SC (衬线，传统文化感)
- **正文/UI**: Noto Sans SC (无衬线，现代清晰)
- **英文**: Inter (与中文搭配良好)

### 组件规范 (shadcn 风格)

- **卡片**: `bg-white rounded-xl border border-stone-200 shadow-sm`
- **按钮 Primary**: `bg-red-600 text-white rounded-lg px-6 py-2.5 hover:bg-red-700 transition-colors`
- **按钮 Secondary**: `bg-stone-100 text-stone-700 rounded-lg px-4 py-2 hover:bg-stone-200`
- **输入框**: `bg-white border border-stone-300 rounded-lg px-4 py-2.5 focus:ring-2 focus:ring-red-500/20 focus:border-red-500`
- **标签**: `bg-red-50 text-red-700 rounded-full px-3 py-1 text-sm`

### 交互说明

1. **每日推荐**: 基于日期种子随机选取，localStorage 记录已展示 ID，循环后重置
2. **换一条**: 点击随机切换（排除当前），带淡入动画
3. **搜索**: 实时过滤，中英文均可搜索
4. **分类筛选**: 点击标签过滤对应章节
5. **中英切换**: 全局切换显示语言，默认双语显示
6. **响应式**: 移动端单列，平板双列，桌面三列卡片网格

### 页面结构

1. **Navbar**: 固定顶部，Logo + 标题 + 语言切换
2. **Hero Section**: 每日推荐语录，大字展示，居中布局
3. **Quotes Grid**: 全部语录卡片网格，含搜索和分类筛选
4. **Footer**: 版权信息，简洁
